Apicación web programada en C# en .Net Core MVC, diseñada para administración eficiente de ventas. Sus funcionalidades son gestión de productos, de ventas, gestión de usuarios y respectivos roles, conexión a base de datos mediante Entity Framework Core, carrito de compras, etc. Fue desarrollada durante el curso de Udemy "ASP.NET Core MVC: Configuración e Implementación (.NET 6)", en el cual se implementan también herramientas como SweetAlert, Syncfusion Grid y DataTable.
